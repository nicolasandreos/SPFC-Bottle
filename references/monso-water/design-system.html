Icon Color Inheritance
All icons in this system are SVG files embedded via <img> tags with fixed internal SVG colors. The check icon uses #95dec9 (aquamarine) as background fill. The cross icon uses a neutral/red fill. Container background and border-radius are controlled by the wrapper class.
